# Crius

Crius is an early prototype for a stock analysis web app. It helps users search mock stocks, view a simple educational stock rating, compare risk levels, and browse a starter watchlist.

## Current Status

Crius is currently a mobile-first website prototype. It is designed to look and feel like a modern fintech app, but it is not a native App Store app yet.

The first version uses mock data only for:

- AAPL
- NVDA
- MSFT
- TSLA

## How To Run

Open `index.html` directly in your browser.

You can also use the VS Code Live Server extension:

1. Open this folder in VS Code.
2. Right-click `index.html`.
3. Choose `Open with Live Server`.

## Tech Used

- HTML
- CSS
- JavaScript

## Disclaimer

Crius currently shows educational mock ratings only. It is not financial advice, and it does not connect to a real stock API.
