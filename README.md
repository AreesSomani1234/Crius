# Crius

Crius is an early prototype for a personalized stock intelligence dashboard. It helps users review a mock watchlist, explore AI-curated trending stocks, and read simplified market article summaries.

## Current Status

Crius is currently a full-screen web dashboard prototype. It uses mock data only and is meant to feel like a modern finance and AI research product.

## Pages Included

- `index.html` - dashboard homepage with hero, watchlist snapshot, trending stocks, and article previews
- `pages/watchlist.html` - searchable full watchlist
- `pages/hot-stocks.html` - searchable AI-curated trending stocks with mock favorite toggles
- `pages/articles.html` - AI-simplified article feed with expandable summaries

The prototype includes mock data for:

- AAPL
- NVDA
- MSFT
- TSLA
- AMZN
- META
- AMD
- PLTR
- GOOGL
- NFLX

## How To Run

Use the VS Code Live Server extension:

1. Open this folder in VS Code.
2. Right-click `index.html`.
3. Choose `Open with Live Server`.

You can also open `index.html` directly in your browser because the project uses plain HTML, CSS, and JavaScript.

## Disclaimer

Crius currently shows educational mock ratings only. It is not financial advice, and it does not connect to a real stock API.
